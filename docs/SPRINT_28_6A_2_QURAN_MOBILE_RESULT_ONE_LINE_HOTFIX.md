# Sprint 28.6A.2 Hotfix — Quran Mobile Search Result One Line

## Objective

Keep mobile Quran navigator Surah search results inside one row.

## Fix

- Arabic Surah name no longer moves to a second line.
- Result button uses 3 columns: number, Surah text, Arabic name.
- Long text is truncated using ellipsis.
- On very narrow phones, secondary metadata is hidden to preserve one-line layout.
