# Sprint 28.5I — Contrast Hard Reset + Serious Learning Rows

## Goal

Readable theme first. NOOR must not use bright text on bright backgrounds or dark text on dark backgrounds.

## UI Rule

Every learning section must show:

1. Clear icon
2. Title
3. Short summary
4. Preview chips
5. One clean expand arrow
6. Hidden detail until the user expands

## Acceptance Checklist

- [ ] Text is readable on every page.
- [ ] Expanded content uses white panels and dark text.
- [ ] Topic rows do not visually collide.
- [ ] Explore copy sounds serious and professional.
- [ ] No bright-on-bright or dark-on-dark UI remains in Today, Explore, Learn and Quran screens.
